📚 Library Management System (Salesforce)

 🚀 Project Overview
This project is a Library Management System built on Salesforce using Lightning Web Components (LWC) and Apex.  
It provides a complete solution to manage books, members, borrowing activities, reservations, and fines with a modern UI and analytics dashboard.



 🔧 Features

 📘 Book Management
- Add, update, and manage books
- Track availability (Available, Borrowed, Reserved)
- View book details and borrow history

 👥 Member Management
- Manage member profiles
- Track borrowing activity
- View member-specific statistics

 🔄 Borrow & Return System
- Issue and return books
- Track due dates
- Automatic overdue detection

 📌 Reservation System
- Reserve books
- Manage reservation status (Active, Completed, Cancelled)

 💰 Fine Management
- Calculate fines based on overdue days
- Display total fines and payment status

 📊 Member Dashboard (Analytics)
- Borrowed, Returned, Reserved, Overdue counts
- Reading interest (category-wise)
- Favorite books
- Borrowing trends over time



 🛠️ Tech Stack
- Salesforce LWC (Lightning Web Components)
- Apex
- SOQL
- Lightning Experience UI



 🧠 Key Highlights
- Custom-built UI using LWC (not standard layouts)
- Real-time data fetching using Apex
- Business logic implementation (fine calculation, overdue tracking)
- Interactive dashboards with charts and insights
- End-to-end system covering both admin and user (member) sides
